stages in working with body parser

  1. Install npm package from terminal-npm install body-parser --save
  2. Embed the bolded lines onto the server.js
            var express = require('express');
        var bodyParser = require('body-parser');   //embeding the package

        var app = express();

        app.use(bodyParser.urlencoded({ extended: false }));   //add this line
        app.use(bodyParser.json());  //add this line

In General

Body-parser extracts the entire body portion of an incoming request stream and exposes it on req.body as something that is easier to work with.

bodyParser.urlencoded()parses (fancy word to convert) the body portion as URL encoded data (which is how browsers tend to send forms) and exposes the resulting object (containing the keys and values) on req.body

bodyParser.json()parses the body portion as JSON and exposes the resulting object onreq.body

we need to put these two app.use functions above all the routes so our req.body is properly populated before the route function get's called
